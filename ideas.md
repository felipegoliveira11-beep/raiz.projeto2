# Brainstorm de Design - Raiz Expedições

## Contexto
Site para agência de turismo de aventura e expedições focado em experiências autênticas na natureza. O design deve transmitir confiança, aventura, autenticidade e conexão com a natureza.

## Abordagem 1: Modernismo Orgânico com Tipografia Assertiva

**Design Movement:** Modernismo Orgânico + Swiss Style Contemporâneo

**Core Principles:**
- Formas fluidas e naturais combinadas com grid estruturado
- Tipografia em contraste: display bold para títulos, sans-serif limpa para corpo
- Hierarquia clara através de escala e peso tipográfico
- Minimalismo inteligente com detalhes orgânicos

**Color Philosophy:**
- Paleta terrestre: verde floresta profundo (#2D5016), terra queimada (#8B4513), areia clara (#F5E6D3)
- Acentos em laranja natural (#E07A5F) para energia e movimento
- Fundo em branco puro com uso estratégico de cores
- Emoção: enraizamento, autenticidade, calor humano

**Layout Paradigm:**
- Hero assimétrico: imagem grande à direita, texto fluido à esquerda
- Seções com diagonal cuts usando clip-path para criar movimento visual
- Grid 12 colunas com espaçamento generoso
- Alternância entre full-width e contained sections

**Signature Elements:**
- Ícones customizados em estilo line-art com traços orgânicos
- Padrão de folhas/ramos sutis como textura de fundo
- Setas diagonais conectando seções (movimento de expedição)
- Cartões com sombra suave e borda esquerda em cor de acento

**Interaction Philosophy:**
- Hover states com mudança de cor e elevação suave
- Cliques revelam mais informações com fade-in
- Scroll triggers animações de entrada de elementos

**Animation:**
- Entrada de elementos com fade + slide suave (200ms)
- Hover em cards: elevação (2px) + mudança de cor de fundo
- Transições de página com dissolve suave
- Parallax sutil em imagens de fundo

**Typography System:**
- Display: "Playfair Display" (bold, 48-72px) para títulos principais
- Headings: "Poppins" (600-700, 24-36px) para seções
- Body: "Inter" (400-500, 16px) para texto corrido
- Accent: "Poppins" (600, 14px) para CTAs e labels

## Abordagem 2: Adventurismo Fotográfico com Tipografia Geométrica

**Design Movement:** Fotojornalismo + Constructivismo Moderno

**Core Principles:**
- Fotografia como protagonista principal
- Tipografia geométrica e ousada para contraste
- Composição assimétrica e dinâmica
- Narrativa visual através de imagens de impacto

**Color Philosophy:**
- Cores vibrantes extraídas das fotos: azul céu (#1E88E5), verde jungle (#388E3C), dourado (#FFA500)
- Fundo em cinza escuro (#2C2C2C) para dar destaque às imagens
- Texto em branco puro para máximo contraste
- Emoção: adrenalina, exploração, descoberta

**Layout Paradigm:**
- Full-width image sections com texto sobreposto
- Masonry layout para galeria de destinos
- Tipografia grande e ousada sobre imagens
- Seções alternadas entre imagem-texto e texto-imagem

**Signature Elements:**
- Badges geométricos (círculos, hexágonos) com informações
- Linhas diagonais cortando o layout
- Numeração em estilo tipográfico (01, 02, 03)
- Overlays de cor semi-transparentes sobre imagens

**Interaction Philosophy:**
- Imagens revelam informações ao hover
- Cliques expandem galerias em lightbox
- Scroll ativa revelação progressiva de conteúdo

**Animation:**
- Zoom sutil em imagens ao hover (1.05x)
- Fade de overlay ao interagir
- Entrada de texto com slide from bottom (300ms)
- Transições de página com wipe diagonal

**Typography System:**
- Display: "Bebas Neue" (700, 64-80px) para títulos
- Headings: "Montserrat" (700, 28-40px) para seções
- Body: "Open Sans" (400, 16px) para descrições
- Accent: "Montserrat" (600, 14px) para CTAs

## Abordagem 3: Minimalismo Elegante com Tipografia Refinada

**Design Movement:** Minimalismo Nórdico + Luxo Discreto

**Core Principles:**
- Simplicidade extrema com sofisticação
- Espaço negativo generoso
- Tipografia refinada como elemento principal
- Paleta reduzida e harmoniosa

**Color Philosophy:**
- Paleta monocromática: preto (#1A1A1A), branco (#FFFFFF), cinza neutro (#808080)
- Acento único em verde musgo (#6B8E6F) para elementos interativos
- Fundo em branco com seções em cinza muito claro (#F9F9F9)
- Emoção: sofisticação, confiança, exclusividade

**Layout Paradigm:**
- Simetria com quebras estratégicas de assimetria
- Muito espaço em branco entre elementos
- Tipografia grande e respeitosa
- Seções bem definidas com separação clara

**Signature Elements:**
- Linhas horizontais delicadas como divisores
- Ícones em estilo minimalista (2px stroke)
- Tipografia como decoração (lettering)
- Cartões com apenas borda sutil

**Interaction Philosophy:**
- Transições suaves e discretas
- Hover com mudança de opacidade
- Cliques revelam detalhes sem dramaticidade

**Animation:**
- Fade suave em todos os elementos (150ms)
- Hover com mudança de opacidade (10-20%)
- Scroll ativa fade-in de elementos
- Transições de página com dissolve elegante

**Typography System:**
- Display: "Cormorant Garamond" (600, 52-68px) para títulos
- Headings: "Lato" (400, 28-36px) para seções
- Body: "Lato" (300, 16px) para texto corrido
- Accent: "Lato" (600, 13px) para CTAs

</response>

## Decisão Final

**Abordagem Selecionada: Modernismo Orgânico com Tipografia Assertiva**

Esta abordagem equilibra perfeitamente os valores da Raiz Expedições: autenticidade (formas orgânicas), profissionalismo (grid estruturado), e energia (tipografia assertiva). A paleta terrestre conecta com a natureza, enquanto a tipografia em contraste garante legibilidade e impacto visual. O layout assimétrico com diagonal cuts cria movimento e dinamismo, refletindo a natureza aventureira das expedições.
